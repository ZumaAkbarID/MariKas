# MariKas

MariKas adalah aplikasi KAS berbasis website yang terintegrasi dengan Bot WhatsApp, Webhook Discord, serta payment gateway Tripay. Website ini digunakan oleh organisasi tertentu & tidak untuk publik.

## Teknologi Utama

```diff
+ Laravel 9
+ PHP 8.1
+ MySQL 8
+ Bootstrap 5
+ Jquery 3.6
+ Fontawesome 5
+ Boxicons
```
